# npm 工作流

# 创建项目

npm init

可以使用 npm init -f（意指 --force，或者使用 --yes）告诉 npm 直接跳过参数问答环节，快速生成 package.json。

初始化 package.json 时的字段默认值是可以自己配置的。

npm config set init.author.email "wangshijun2010@gmail.com"
npm config set init.author.name "wangshijun"
npm config set init.author.url "http://github.com/wangshijun"
npm config set init.license "MIT"
npm config set init.version "0.1.0"

# 运行多个命令

# 串行

让多个 npm script 串行的典型用例，只需要用 && 符号把多条 npm script 按先后顺序串起来即可。

{
  "scripts": {
    "lint:js": "eslint *.js",
    "lint:css": "stylelint *.less",
    "test": "mocha tests/",
    "start": "npm run lint:js && npm run lint:css && npm run test"
  }
}

npm start

串行执行的时候如果前序命令失败（通常进程退出码非0），后续全部命令都会终止。

# 并行

从串行改成并行，把连接多条命令的 && 符号替换成 & 即可。

可以使用 npm-run-all 实现更轻量和简洁的多命令运行。文档

npm install npm-run-all -D

{
  "scripts": {
    "start": "npm-run-all lint:js lint:css npm run test"
  }
}

# npm script 传递参数

{
  "scripts": {
    "new": "node ./build/new.js $*"
  }
}

npm run new p1 p1
