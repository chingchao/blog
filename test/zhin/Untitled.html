# markdown-it-anchor 路径错误问题

markdown-it-anchor 插件给标题标签生成一个 a 标签，默认情况 href 值为 '#' + 标题的 id，如果项目使用 hash 路由，url 上原本就带有哈希值，点击 a 标签，浏览器会直接将哈希值替换。anchor was not added to correct path

<h2 id="anchor">
  <a href="#anchor"></a>
</h2>

如果当前 url 为 localhost:8080/#/component，点击 a 标签后，url 会变成 localhost:8080/#/anchor，期望的结果是 localhost:8080/#/component/#anchor。

第一种解决办法是自定义 permalinkHref 函数，然后再实现页面滚动逻辑，不过所有锚点拼接上 #/component/，路由使用会有问题。

- const permalinkHref = slug => `#${slug}`
+ const permalinkHref = slug => `#/component/#${slug}`

第二种方法是直接修改渲染后的锚点 href 值，在前面拼接上当前 url。滚动到指定标题，可以使用 scrollIntoView 实现。

export default {
  	mounted() {
        this.resetAnchorHref();
        this.scrollToAnchor();
    },
    beforeRouteUpdate(to, from, next) {
        next();
        setTimeout(() => {
            if (to.path !== from.path) {
                this.resetAnchorHref();
            }
            if (to.path === from.path && to.hash) {
                this.scrollToAnchor();
            }
        }, 100);
    },
    methods: {
        // 修改 href
        resetAnchorHref() {
            const anchors = document.querySelectorAll('h1 a, h2 a, h3 a, h4 a, h5 a');
            anchors.forEach(a => {
                const href = a.getAttribute('href');
                a.href = location.href.split('#').slice(0, 2).join('#') + href;
            });
        },
        // 滚动
        scrollToAnchor() {
            const hash = this.$route.hash.match(/#([^#]+)$/);
            if (!hash) return;
            const elm = document.querySelector(hash[0]);
            if (!elm) return;
            elm.scrollIntoView({
                behavior: 'smooth'
            });
        }
    }
}

# 文件写入插件 file-save

使用 fs.writeFileSync()，如果路径带有一个不存在的文件夹名称，会写入文件失败。

const fs = require('fs');
fs.writeFileSync('./dir/f.js', '');
// no such file or directory, open './dir/f.js'

递归创建文件夹可以直接使用 fs.mkdir(path, opt)，opt.recursive 设置成 ture，类似于 mkdir -p 命令。

const fs = require('fs');
const dir = path.dirname(file-save);
fs.mkdir(dir, {
  recursive: true
});

写入文件需要先创建文件夹，通过插件可以简化，比如使用 file-save。

const fileSave = require('file-save');
fileSave('./dir/f.js').write('');
